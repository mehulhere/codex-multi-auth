## Core

### Improvements

- Runtime rotation is now enabled by default; opt out via `codex auth rotation disable`, `codexRuntimeRotationProxy=false`, or `CODEX_MULTI_AUTH_RUNTIME_ROTATION_PROXY=0`.
- Added best-effort package auto-update checks for installed users, skipped in CI, tests, and local worktrees.
- Updated live quota probes and fallback model defaults to current selectors including `gpt-5.3-codex` and `gpt-5.5`.
